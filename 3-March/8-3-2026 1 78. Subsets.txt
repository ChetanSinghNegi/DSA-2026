Time Complexity
At each index, the recursion makes 2 choices — exclude or include the element.
For n elements, this creates a binary recursion tree with 2^n leaf nodes (each leaf = one subset).
At each leaf, new ArrayList<>(subset) copies the subset → O(n) cost.
Total = 2^n × O(n) = O(2^n × n)
Space Complexity
subsets listO(2^n × n)Stores all 2^n subsets, each up to size nRecursion call stackO(n)Tree depth is exactly n (idx goes 0 → n)subset listO(n)Holds at most n elements at a time
Time:  O(2^n × n)
Space: O(2^n × n)

M1) 
class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> subsets = new ArrayList<>();
        helper(0, new ArrayList<>(), subsets,nums);
        return subsets;
    }
    public void helper(int idx, List<Integer> subset, List<List<Integer>> subsets, int[] nums){
        if(idx == nums.length) {
            List<Integer> base = new ArrayList<>(subset);
            subsets.add(base);
            return;
        }
        helper(idx+1, subset, subsets, nums);
        subset.add(nums[idx]);
        helper(idx+1, subset, subsets, nums);
        subset.remove(subset.size()-1);
    }
}

class Solution {
    public List<List<Integer>> subsets(int[] nums) {
        List<List<Integer>> subsets = new ArrayList<>();
        helper(0, new ArrayList<>(), subsets,nums);
        return subsets;
    }
    public void helper(int idx, List<Integer> subset, List<List<Integer>> subsets, int[] nums){
        subsets.add(new ArrayList<>(subset));
        for(int i = idx; i< nums.length;i++){
            subset.add(nums[i]);
            helper(i+1, subset, subsets, nums);
            subset.remove(subset.size()-1);
        }
    }
}
